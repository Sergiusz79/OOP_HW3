Описание Классов, функций и методов

Game - интерфейс (Интерфейс это конструкция языка Java, в рамках которой принято описывать абстрактные публичные
    ( abstract public ) методы и статические константы ( final static ). С помощью интерфейса можно указать,
    что именно должен выполнять класс его реализующий, но не как это делать. Способ реализации выбирает сам класс.
    В широком смысле интерфейс — некий «пульт», который связывает две взаимодействующие друг с другом стороны.

AbstractGame implements Game - используется Implements для указания того, что класс или структура
    реализует один или несколько интерфейсов. Начальное значение gameStatus - INIT.

    generateWord - генератор слова, использует generateCharList, который генерирует набор значений(GameNumber -
    генерирует набор чисел, RUGame - генерирует алфавит ru, ENGame - генерирует алфавит en), рандомно добавляет
    из набора значения формируя загаданное "слово".

    start - принимает на вход длину "слова" sizeWord и к-во попыток его отгадать maxTry, запускает метод generateWord(),
    меняет значение переменной gameStatus с INIT на START, и выводит на печать подсказку.

    Answer inputValue(String value) - основная логика сравнения загаданного "слова" и введённого пользователем,
    подсчёт быков и коров. В зависимости от результата - изменение статуса WINNER или LOSE.
    Возврат - введённого пользователем "слова", к-ва быков и коров.

enum GameStatus - (Интерфейс Enumeration) перечисление, список именованных констант. Определяет методы,
    с помощью которых вы можете перечислить (получить по одному) элементы в коллекции объектов.

public class Answer - содержит поля: userInput, bull, cow, геттеры и сеттеры, метод toString и конструктор.
